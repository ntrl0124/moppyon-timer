# 文鳥休憩タイマー 「もっぴょんタイマー」
 
文鳥のもなかがぴょんぴょん跳ねて、休憩時間をお知らせします。
一定間隔で休憩を促すオーバーレイを表示する、macOS 向けのデスクトップアプリです
 
 
## ■ 使い方
 
1. アプリを起動すると、設定画面が開きタイマーが自動でスタートします。
2. デフォルトでは30分後に休憩オーバーレイが表示されます。（設定で変更可能）
3. 休憩が終わったら「作業に戻る」または「閉じる」ボタンでオーバーレイを閉じてください。
4. 閉じると、次の休憩タイマーが自動でスタートします。
 
メニューバーのアイコンから、以下の操作もできます。
- 一時停止 / 再開
- 次の休憩をスキップ
- 今すぐ休憩を表示
 
 
## ■ 設定
 
設定画面では以下を変更できます。
 
- 休憩の間隔
- 休憩（オーバーレイ表示）時間
- 音声を鳴らすかどうか
- 起動時にタイマーを自動スタートするかどうか。
 
 
## ■ 初回起動時の警告について
 
初回起動時に「開発元を確認できないため開けません」と表示される場合があります。
その場合は以下の手順で起動してください。
 
1. Finder でアプリを右クリック
2. 「開く」を選ぶ
3. ダイアログでもう一度「開く」を選ぶ
